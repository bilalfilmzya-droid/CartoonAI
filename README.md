# 🎨 CartoonAI - Image to Cartoon Converter

Apni photos ko 6 cartoon styles mein convert karo!

## 📱 App Features
- 💥 Comic Pop — Bold posterized comic style
- ✨ Anime — Smooth anime with outlines  
- ✏️ Sketch — Pencil edge drawing
- 🎨 Watercolor — Soft painted look
- 🌈 Neon — Vivid neon colors
- 📺 Retro — Limited vintage palette
- Gallery se photo choose karein
- Cartoon image save ya share karein

---

## 🚀 APK Kaise Banayein (Sirf 3 Steps!)

### Step 1: GitHub Par Upload Karo
1. github.com par naya **repository** banao
2. Is ZIP ko extract karo
3. Poora `CartoonAI-Android` folder ka content upload karo

### Step 2: Build Run Karo
- Repository mein **Actions** tab par jaao
- **"Build CartoonAI APK"** workflow click karo
- **"Run workflow"** button dabao
- ~5 minute wait karo ⏳

### Step 3: APK Download Karo
- Build complete hone par workflow par click karo
- **Artifacts** section mein **CartoonAI-APK** download karo
- APK file apne Android phone par install karo ✅
